## PIP安装

- 获取 pip  安装程序

```
wget https://bootstrap.pypa.io/get-pip.py

```
- 下载安装pip
```
sudo python get-pip.py
```
```
--no-cache-dir # 忽略之前的cache
-i # 指定源
-r # 指定requirements文件进行安装
```
